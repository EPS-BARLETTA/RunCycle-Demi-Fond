RunCycle — Aide

Guide rapide Prof & Élèves

👨‍🏫 Prof — Créer une séance

  1. Ouvrez Mode Prof.
  2. Choisissez une séance type (S1–S5) ou ajustez les paramètres.
  3. Réglez la piste (200/250/300/333/400 m).
  4. Cliquez Générer code. Un code court est créé pour les présélections, un code étendu pour une séance perso.
  5. Donnez le code aux élèves (le QR séance est facultatif).

👩‍🎓 Élèves — Saisir et transmettre

  1. Ouvrez Mode Élèves en binôme.
  2. Saisissez vos profils (Nom, Prénom, Classe, Sexe, VMA).
  3. Entrez le code donné par le prof.
  4. Après la course, saisissez vos distances par bloc et votre RPE (0–10).
  5. Affichez le QR DUO pour que le prof le scanne dans ScanProf (ou QR A/B).

ℹ️ Détails

  • Individualisation : la cible par bloc est calculée avec votre VMA × %VMA.
  • Codes : 4 caractères pour les séances type, 8 caractères pour les séances perso (stockées localement).
  • Offline : tout fonctionne sans serveur ; les séances perso sont disponibles sur l’appareil qui les a créées.
