Aide

Principe

Le mode Prof permet de créer/choisir une séance (6 modèles + personnalisé) et de générer un code de séance à donner aux élèves (ou un lien direct). Tout est encodé localement (pas de serveur).

Le mode Élèves lit ce code, lance le chrono, propose l’interface adaptée (fractionné ou mange plots) et génère un QR final à scanner avec ScanProf.

“Mange plots”

Plots tous les 12,5 m. Choisir une fenêtre (ex. 1:30) et saisir le nombre de plots mangés → l’app calcule la vitesse (km/h). Fenêtres dispo : 1:30, 3:00, 4:30, 6:00.

Code Prof

Le code est vérifié par empreinte SHA‑256 (jamais affiché). Pour le changer, éditez prof.js : variable PROF_HASH.
