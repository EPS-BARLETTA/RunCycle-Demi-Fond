Guide rapide Prof & Élèves

👨‍🏫 Prof — Générer un code séance

  1. Ouvrez Mode Prof (code: 57).
  2. Choisissez une séance type (S1–S5) puis cliquez ↺ Charger séance type.
  3. Cliquez 🔐 Générer / actualiser le code → l’encart “CODE ÉLÈVES” affiche le code à dicter.
  4. Option : modifiez la séance → cliquez à nouveau 🔐 Générer (un code perso est créé et stocké localement).
  5. Option : entrez votre code préféré (champ “Remplacer le code élèves”).

👩‍🎓 Élèves — Utiliser le code

  1. Ouvrez Mode Élèves en binôme.
  2. Entrez le code (ex. S2 = 202B), saisissez VMA A/B et la piste (m).
  3. À la fin : entrez vos distances par bloc + RPE → affichez le QR DUO pour ScanProf.
